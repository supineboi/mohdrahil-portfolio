## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Install dependencies:**
   ```bash
   cd resume-website
   npm install
   ```

2. **Replace the profile image:**
   - Add your professional photo to `src/assets/images/profile.jpg`
   - Recommended size: 512x512px or higher
   - Format: JPG, PNG, or WebP

3. **Add background video (optional):**
   - Add your video to `src/assets/Videos/profile-video.mp4`
   - Recommended: Short loop (5-10 seconds), 1080p or 720p
   - Keep file size under 5MB for fast loading
   - Video will autoplay, loop, and be muted

4. **Update resume data:**
   - Edit `src/data/resumeData.js` with your information
   - Update personal info, skills, experience, education, etc.

5. **Add your resume PDF:**
   - Place your resume PDF in the `public/` folder as `resume.pdf`
   - Or update the download link in `src/sections/Hero.jsx`

### Development

Run the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:3000`

### Build for Production

Create an optimized production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## ğŸ“ Customization Guide

### 1. Update Your Information

Edit `src/data/resumeData.js`:

```javascript
export const personalInfo = {
  name: "Your Name",
  title: "Your Title",
  email: "your.email@example.com",
  phone: "+1234567890",
  // ... update all fields
};
```

### 2. Change Profile Picture

Replace `src/assets/images/profile.jpg` with your photo. The current image is a placeholder generated by UI Avatars.

### 3. Customize Colors

Edit `tailwind.config.js` to change the primary color scheme:

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        // Change these values
        500: '#0ea5e9',
        600: '#0284c7',
        // ...
      },
    },
  },
}
```

### 4. Modify Sections

Add, remove, or reorder sections in `src/App.jsx`:

```javascript
<main>
  <Hero />
  <About />
  <Skills />
  <Experience />
  <Education />
  <Contact />
</main>
```

## ğŸŒ Deployment

### Deploy to Vercel (Recommended)

1. Push your code to GitHub
2. Import your repository on [Vercel](https://vercel.com)
3. Deploy with default settings
4. Your site will be live in seconds!

### Deploy to Netlify

1. Push your code to GitHub
2. Import your repository on [Netlify](https://netlify.com)
3. Build command: `npm run build`
4. Publish directory: `dist`

### Deploy to GitHub Pages

1. Install gh-pages:
   ```bash
   npm install --save-dev gh-pages
   ```

2. Add to `package.json`:
   ```json
   "homepage": "https://yourusername.github.io/resume-website",
   "scripts": {
     "predeploy": "npm run build",
     "deploy": "gh-pages -d dist"
   }
   ```

3. Deploy:
   ```bash
   npm run deploy
   ```
